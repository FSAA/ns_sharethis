Configuration
=============

Quick & Easy "Global" configurataion of Sharethis.com
=====================================================

.. rst-class:: ol-bignums

#. Switch to the module Admin tools *Extensions* and then
   edit **configuration**.

#. Enable Global Interface into this **settings**.
   See these screenshots:

   .. figure:: Images/TYPO3-EXTns_sharethis-Config_1-1.png
      :alt: screenshot

   .. figure:: Images/TYPO3-EXTns_sharethis-Config_1-2.png
      :alt: screenshot


Setup "Particular Page Only" configuration of Sharethis.com
===========================================================

.. rst-class:: ol-bignums

#. Switch to the module Admin tools *Extensions* and then edit
   **configuration** and then **Disable Global Interface** into this
   *settings*.

#. Now add extension Above/Below page as per your requirement.

#. Now select plugin **Nitsan Social widget** and configure it as per
   requirement. See these screenshots:

   .. figure:: Images/TYPO3-EXTns_sharethis-Config_2-1.png
      :alt: screenshot

   .. figure:: Images/TYPO3-EXTns_sharethis-Config_2-2.png
      :alt: screenshot

   .. figure:: Images/TYPO3-EXTns_sharethis-Config_2-3.png
      :alt: screenshot

.. _Clearing-the-cache:

Clearing the cache
==================

Please use the buttons 'Flush frontend caches' and 'Flush general caches'
from the top panel. The 'Clear cache' function of the install tool will also
work perfectly.
